# AI Health Companion for Rural Communities  

A simple AI-powered mobile app that helps villagers share health problems through **voice or text** and get easy guidance in local languages.  

---

## Problem Statement  
Rural communities often lack doctors, face literacy barriers, and struggle to describe symptoms. Small health issues become serious due to delays, and ASHA workers lack real-time tools to guide patients.  

---

## Our Solution  
- Villagers can **speak or type** symptoms in their own language  
- AI **understands symptoms** and suggests possible causes  
- Provides **simple cures, first-aid tips, or doctor alerts**  
- Works **offline** and supports **ASHA workers** in the field  

---

## Tech Stack & Azure  
- **Mobile App:** Flutter / React Native  
- **AI:** Azure Cognitive Services + Azure ML  
- **Database:** Azure SQL / Cosmos DB (free tier)  
- **Backend:** Node.js / Python on Azure App Service  
- **Dashboard:** Power BI (free tier)  

---

## Key Benefits  
- Accessible healthcare for villages  
- Supports local languages and offline mode  
- Empowers ASHA workers with quick guidance
